Classifier Free Guidance for Diffusion models

Posted on 2024-11-20 Edited on 2025-06-06 In Generative models

Classifier Guidance

Diffusion models predict score functions ∇xlog⁡p(x)\nabla_x \log{p(x)}∇x​logp(x). If the input contains conditional information such as class label, text, or image embeddings, the new score function is ∇xlog⁡p(x∣y)\nabla_x \log{p(x|y)}∇x​logp(x∣y). By Bayes rule, the new score function can be written as

∇xlog⁡p(x∣y)=∇xlog⁡p(y∣x)+∇xlog⁡p(x).\nabla_x \log{p(x|y)} = \nabla_x \log{p(y|x)} + \nabla_x \log{p(x)}. ∇x​logp(x∣y)=∇x​logp(y∣x)+∇x​logp(x).

From here we notice that ∇xlog⁡p(y∣x)\nabla_x \log{p(y|x)}∇x​logp(y∣x) is exactly the log gradient of a discriminative model, or we can call it a classifier. Given a unconditional diffusion model that predicts ∇xlog⁡p(x)\nabla_x \log{p(x)}∇x​logp(x), and a classifier that allows us to compute ∇xlog⁡p(y∣x)\nabla_x \log{p(y|x)}∇x​logp(y∣x), we have the formulation for Classifier Guidance:

∇xlog⁡p(x∣y)=∇xlog⁡p(x)+γ∇xlog⁡p(y∣x),\nabla_x \log{p(x|y)} = \nabla_x \log{p(x)} + \gamma\nabla_x \log{p(y|x)}, ∇x​logp(x∣y)=∇x​logp(x)+γ∇x​logp(y∣x),

where γ\gammaγ is the guidance scale.

However, it is expensive to train another classifier. Can we guide our diffusion model without additional training?

Classifier Free Guidance

Noticing that we have

∇xlog⁡p(x∣y)=∇xlog⁡p(y∣x)+∇xlog⁡p(x),∇xlog⁡p(y∣x)=∇xlog⁡p(x∣y)−∇xlog⁡p(x).\begin{aligned} \nabla_x \log{p(x|y)} = & \nabla_x \log{p(y|x)} + \nabla_x \log{p(x)}, \\ \nabla_x \log{p(y|x)} = & \nabla_x \log{p(x|y)} - \nabla_x \log{p(x)}. \end{aligned}∇x​logp(x∣y)=∇x​logp(y∣x)=​∇x​logp(y∣x)+∇x​logp(x),∇x​logp(x∣y)−∇x​logp(x).​

Plug this back into the formula of Classifier Guidance:

∇xlog⁡p(x∣y)=∇xlog⁡p(x)+γ∇xlog⁡p(y∣x),=∇xlog⁡p(x)+γ(∇xlog⁡p(x∣y)−∇xlog⁡p(x)),=(1−γ)∇xlog⁡p(x)+γ∇xlog⁡p(x∣y).\begin{aligned} \nabla_x \log{p(x|y)} = &\nabla_x \log{p(x)} + \gamma\nabla_x \log{p(y|x)}, \\ = & \nabla_x \log{p(x)} + \gamma(\nabla_x \log{p(x|y)} - \nabla_x \log{p(x)}), \\ = & (1-\gamma)\nabla_x \log{p(x)} + \gamma\nabla_x \log{p(x|y)}. \end{aligned}∇x​logp(x∣y)===​∇x​logp(x)+γ∇x​logp(y∣x),∇x​logp(x)+γ(∇x​logp(x∣y)−∇x​logp(x)),(1−γ)∇x​logp(x)+γ∇x​logp(x∣y).​

Normally, we train a conditioned diffusion model with an additional Null class embedding, which corresponds to the unconditional diffusion case. Thus, we only need to train the model once with true class embeddings randomly replaced by this Null class embedding. Physical intuition

When γ=1.0\gamma=1.0γ=1.0, our moving direction is ∇xlog⁡p(x∣y)\nabla_x \log{p(x|y)}∇x​logp(x∣y).
When γ>1.0\gamma>1.0γ>1.0, the model is pushing our particles away from other modalities and faster to arrive at the distribution of class conditioned data.
