Data-driven Discovery of PDEs

Posted on 2025-10-27 Edited on 2025-10-30 In Paper notes

Problem

Long Rollouts of Neural Operators (NOs) accumulate errors over time, leading to significant deviations from the true solution.

Physics-informed Temporal Adaptation (PITA)

Given a base model, finetune with a fitted weight vector and library matrix to approximate the g.t. PDE equation.

Assume the base model gives a prediction U(θ)U(\mathbf{\theta})U(θ), the polynomial library matrix Φ(U)\Phi(U)Φ(U) is constructed from UUU and its spatial derivatives:

Φ(U)=[1,U,U2,…,Ux,UUx,Uxx,…]\Phi(U) = [1, U, U^2, \ldots, U_x, U U_x, U_{xx}, \ldots] Φ(U)=[1,U,U2,…,Ux​,UUx​,Uxx​,…]

Thoughts

This method requires precomputed derivative terms from the base model’s prediction. The library needs to be carefully designed to include the hypothesized terms of the underlying PDE. When the true PDE is unknown, selecting an appropriate library can be challenging and may require domain expertise.
